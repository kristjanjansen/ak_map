How many books?

This visualization shows how many books and publications are in Estonian public libraries (per parish).

Data: For map contours: Estonian Land board parish map. For book data: Statistics Estonia cultural maps.

Tools: Mapshaper for geometry simplification, GDAL for projection fixing, Kartograph for SVG conversion and display, Miso Dataset for data manipulation, Chroma.js for color manipulation.

Code: Github

Team: A joint project between Kuressaare Training Centre data visualization course, Open Data Estonia and Estonian Open Map Society.
